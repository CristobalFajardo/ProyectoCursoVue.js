## Requisitos

* Node 8
* Git
* Algun editor como VS

## Comando de instalación

Clone the repo and install the dependencies.

```bash
npm i
```

## Comando para ver en explorador

```bash
npm run dev
```

```bash
npm run serve
```

Open [http://localhost:8080]
